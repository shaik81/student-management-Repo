
# Student Management System


## Overview

The Student Management System is a backend-focused application designed to streamline the management of student records in educational institutions. The system provides secure and efficient student data management through CRUD operations, authentication mechanisms, and role-based access control.

Built using Java and PostgreSQL, the application demonstrates modern backend development practices, including RESTful API design, secure authentication, database management, and user authorization.

## Features

### Student Record Management

* Create, view, update, and delete student records.
* Manage student information efficiently through REST APIs.
* Search and retrieve student details from the database.

### Authentication & Security

* JWT (JSON Web Token) Authentication for secure access.
* OAuth2 Integration for enhanced authentication support.
* Password Encoding for secure credential storage.
* Session Management to maintain authenticated user sessions.

### Authorization

* Role-Based Access Control (RBAC).
* Different user roles with controlled access to system resources.
* Secure endpoint protection based on user permissions.

### Data Validation

* Input validation to ensure data integrity.
* Error handling and exception management for reliable application behavior.

### Database Management

* PostgreSQL integration for persistent data storage.
* Efficient database operations and relationship management.
* Database administration using DBeaver and pgAdmin.

### API Development & Testing

* RESTful API architecture.
* API testing and validation using Postman.
